Web AR/AR.js Demos

Print out this marker image then click on one of the demo's and point your camera at it.

Transparent PNG

A transparent PNG. This is a super simple demo. There is almost nothing to it. These are the default settings except for the sizing, which was specified to avoid distorting the image.

Marker: Default

Transparent PNG: Floating

Same concept as above but the image is floating higher above the marker. You may need to back away from the marker a bit.

Marker: Default

Curved Transparent PNG

This uses the a-curvedimage element instead of the a-image element.

Marker: Default

A Full Scene

This is loading a glTF model and rendering it on top of the marker. This is pretty rad but also requires the most work because you have to make the 3D model.

You can read more about using glTF models in a-frame here.

Marker: Default

Caveats
    jli>Marker must be clearly visible and not distorted. Even slight curves are enough to skew detection.
